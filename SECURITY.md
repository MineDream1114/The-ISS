# Security Policy

## Supported Versions

This repository contains static HTML for a PYP Exhibition project about the International Space Station (ISS). Because this is an educational research project, we only maintain the latest version of the site.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Security is important to us. Even though this is a simple HTML website about the ISS, we want to ensure it is safe for all visitors.

If you discover any security vulnerabilities or issues (such as broken links or script errors), please do not open a public issue.

We will review the issue and update the HTML code as quickly as possible.
